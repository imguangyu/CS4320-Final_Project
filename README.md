# Tech Stack

- Microservices: Node.js


- Database: mongodb
- Framework: Vue.js
- CSS: Bootstrap
- Http Library: Axios
- Agile methodology: Scrum in Axosoft 

# Quick Start

``` bash
1. intall environment for node.js and mongodb.
2. npm install
3. start mongo server at port 27017 
4. node napp.js
5. Open login.html in your browser
```
